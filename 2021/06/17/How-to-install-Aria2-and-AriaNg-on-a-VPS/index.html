VPS搭建Aria2与AriaNg教程

发表于 2021-06-17 更新于 2021-09-10 Valine：
本文字数： 1.1k

Aria2是个好东西，尤其是搭载在一台VPS上。

很久没有更新博客了，上次的技术教程还是Typecho博客的迁移笔记，这段时间事情比较多，自己懒癌又犯了上来，很快把博客的事情忘光了，想起来的时候我就在码这篇教程了。

我想了一下，关于技术教程这方面的东西我是没什么可以写的了，毕竟我是个医学生，还是临床医学专业，这个专业和IT技术基本八竿子打不着，如果大家想单纯看技术教程的话，我不建议你在这里停留太久（苦笑ing

以后尽量克服懒癌吧。

关于Aria，大家应该略有耳闻，速度快，占用小，不限速，且在Windows、MacOS、Linux下都可以安装，树莓派、NAS、路由器等设备也可以安装，支持 HTTP / FTP / BT / Magnet 磁力链接等类型的文件下载。结合第三方扩展甚至可以破解百度网盘的速度限制，但是这得看百度了，很可能手中的扩展或脚本很快就失效了，费时费力，几个人合租一下一年就40多点，这玩意也就是个下载器。

这次的Aria我是在VPS上安装的，毕竟25G的空间还有大带宽，不来BT下载可惜了。

PS：请注意你的VPS服务商是否有版权限制，若有的话进行BT下载很可能终止你的VPS服务！

首先我使用的是CCAA的一键搭建脚本，然后就发现它始终无法连接我的Aria2，当时使用的命令是这些，我也搞不清哪里出错了…..

yum -y install wget unzip curl
wget https://github.com/helloxz/ccaa/archive/master.zip
unzip master.zip && cd ccaa-master && sh ccaa.sh

这里是CCAA的官方帮助文档，我感觉是我的问题，若有大佬了解其缘由欢迎告知我。

但是已经搭建起了AriaNg和filemanager，下面搭建Aria就行了。

搭建Aria2的脚本，Xshell连接VPS，输入命令：

wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/aria2.sh && chmod +x aria2.sh && bash aria2.sh

如图所示，应该是这样的：

将Aria2安装好以后，输入7，修改配置信息，将下载地址改为CCAA的下载地址。

在AriaNg中的设置中更改端口，更改密匙，应该就能连接成功了。

PS：我希望你们不要按照这个教程来了，CCAA的官方教程写的很清楚，我的经历比较曲折，当然如果你遇到了同样问题的话，这个方法不妨一试。
